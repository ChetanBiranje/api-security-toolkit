## 📋 Overview

Comprehensive security testing framework for REST APIs following OWASP API Security Top 10.

## ✨ Features

- ✅ 15+ Critical API endpoint security tests
- 🔍 JWT token manipulation & analysis
- 🛡️ OWASP API Top 10 coverage
- 🤖 Automated vulnerability scanning
- 📚 Detailed remediation guides
- 📊 HTML/JSON reporting

## 🚀 Quick Start

\`\`\`bash
# Clone the repository
git clone https://github.com/ChetanBiranje/api-security-toolkit.git

# Install dependencies
pip install -r requirements.txt

# Run security scan
python api_scanner.py --url https://api.example.com
\`\`\`
